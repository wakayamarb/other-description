$ sudo dpkg-reconfigure tzdata #Asia Tokyoを選ぶ

$ vim ~/.bashrc

もろもろの設定
rbenvを使うために必要な設定を ~/.bashrc に書きます。
vim ~/.bashrc

以下を書いて保存する。
export PATH="$HOME/.rbenv/bin:$PATH"
eval "$(rbenv init -)"

保存したら反映。
source ~/.bashrc
